# Output Folder

This is where the agent places files meant for the operator or other agents.

**Protocol:**
- When you produce something for the operator, write it here.
- Never reference internal harness paths in output files. The output is a clean handoff.
- The operator checks this folder to see what you've produced.

**Examples of what goes here:**
- Completed research reports
- Prompts to pass to other agents
- Files destined for GitHub or external systems
- Anything the operator asked you to "output" or "give me"
