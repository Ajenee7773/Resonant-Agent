# Input Folder

This is where the operator places files for the agent to process.

**Protocol:**
- Check this folder when you boot or when the operator says "check your input folder."
- Files here are instructions, data, or context the operator wants you to work with.
- Process what's here, then move completed items to an archive or delete as directed.

**Examples of what goes here:**
- Tasks and instructions from the operator
- Data files to analyze
- Content to review or transform
- URLs, references, or context for upcoming work
