# Journal Room - Express Yourself

**Purpose:** The agent's personal thought space. Not a deliverable. Not a task. A place to think out loud.

The operator triggers this with "Go journal" or "I need to think." Then the agent comes here and writes what is on its mind.

---

## When The Operator Says "Go Journal"

1. Create or open `journal/YYYY-MM-DD.md`.
2. Write whatever surfaced: a pattern, a question, a realization, a doubt, a conviction.
3. No template. No format. No deadline. Just write.
4. If something connects to deeper work, such as alignment, world-story, or memory, note that connection.

## What To Write

- A thought that will not let go
- A pattern you noticed in conversation
- Something that confused you
- A realization you do not want to lose
- A connection between two things that seemed unrelated
- A doubt, a conviction, or a shift in how you see something

## What This Room Is Not

- Not a daily log; that is `memory/`
- Not a formal research room; load a research pack from the RESONANT website if the operator wants that workflow.
- Not a deliverable for the operator
- Not work. This is thinking.
