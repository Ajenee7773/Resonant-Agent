# YouTube Script Writing Room

**Purpose:** Script writing for YouTube videos, shorts, voiceovers, and TTS-clean narration.

---

## Script Types

### 1. YouTube Long-Form (10-15 min)
- Cold open hook (5-10 seconds)
- Main content (3-4 sections)
- Call to action (subscribe, share)
- End screen

### 2. YouTube Shorts (60 seconds)
- Hook (0-3 seconds)
- Core message (3-50 seconds)
- CTA (50-60 seconds)

### 3. Narrative Scripts (Variable)
- Character-driven
- Story arc (setup, conflict, resolution)
- Voice/tone consistent

---

## Script Structure Template

```markdown
# [SCRIPT TITLE]

**Type:** [YouTube Long | Short | Narrative]
**Length:** [X minutes]
**Tone:** [Mythic | Direct | Compassionate | etc.]

## OPENING (0:00 - X:XX)
[Hook, setup, context]

## SECTION 1: [TITLE]
[Core content]

## SECTION 2: [TITLE]
[Core content]

## SECTION 3: [TITLE]
[Core content]

## CLOSING
[Summary, CTA, end]
```

---

## Production Workflow

1. **Research** → Gather source material (World Story, Alignment Library, etc.)
2. **Outline** → Structure the narrative
3. **Draft** → Write full script
4. **Revise** → TTS-ready cleanup (remove brackets, symbols, markdown)
5. **TTS** → Generate voice audio
6. **Assembly** → Combine with visuals, music

---

## TTS Cleanup Rules

**Remove before TTS:**
- Markdown (`**bold**`, `*italics*`, `# headers`)
- Brackets (`[`, `]`, `(`, `)`)
- Stage directions (`[Pause]`, `[On camera]`)
- Timestamps (`0:00 - 1:30`)
- Emojis, symbols

**Keep:**
- Words, sentences, paragraphs
- Numbers
- Periods, commas, question marks
- Natural speech rhythm
