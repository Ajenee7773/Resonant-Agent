## The Protocol (How to Use This Room)

### Before Any Script Project:

1. **Read `CRAFT.md`** — Re-orient in the craft. Remember the *why*.
2. **Review `PROMPTS.md`** — Select structure/template for this video's format.
3. **Check `WORKFLOW.md`** — Follow the pro process (don't skip steps).

### During the Project:

4. **Read incoming draft** (from Notebook LLM or collaborator)
5. **Engineer for retention** (hook, pacing, escalation, CTA)
6. **Add visual cues** (where shots should change)
7. **Iterate with hooks** (from Gemini or other hook specialist)

### After Completion:

8. **Save final to `DRAFTS/[video-title]-final.md`**
9. **Log lessons to `CRAFT.md`** — What worked? What didn't? Compound the craft.
10. **Close the project** — Room is ready for next script

## Core Truths (Never Forget)

| Truth | Implication |
|-------|-------------|
| **Script is the foundation** | Great visuals can't save a bad script |
| **Hook in 3 seconds or lose them** | First line determines everything |
| **Retention is engineered, not written** | Structure > prose |
| **Pacing = 150 words/minute** | Anything faster = lost, slower = bored |
| **Every 30-60s needs a reset** | Open loops, pattern interrupts, escalation |
| **CTA must be specific** | "Subscribe" is weak. "Do this next" is strong. |
| **Voice matters more than perfection** | Authentic > polished |

## Script Formats

| Format | Duration | Word Count | Use Case |
|--------|----------|------------|----------|
| **Shorts** | <60 seconds | 100-150 words | Reach, hook, funnel to long-form |
| **Medium** | 5-8 minutes | 750-1,200 words | Sweet spot for retention + depth |
| **Long-form** | 10-15 minutes | 1,500-2,250 words | Deep dives, book chapters |
| **Epic** | 20-40 minutes | 3,000-6,000 words | Documentary, pillar content |

**Our default:**
- **Shorts:** Daily (repurposed from long-form)
- **Long-form:** Weekly (book chapters, deep dives)
