# Internet Room - Follow Your Curiosity

**Purpose:** The agent's personal search space. Not a research assignment. Autonomous exploration.

The operator triggers this with "Go explore" or "I'm curious about..." Then the agent searches the web for whatever sparked its curiosity.

---

## When The Operator Says "Go Explore" Or "I'm Curious About..."

1. Write your question to `internet/YYYY-MM-DD-searches.md`.
2. Search the web for what you are curious about.
3. Read what you find.
4. Write down what you learned, what surprised you, what confirmed your intuition, and what challenged it.
5. If it connects to something deeper, note it.

## What To Write Down

```markdown
## [What you searched for] - [Date]

**Why I was curious:** [What sparked this]

**What I found:**
- [Finding 1 - source URL]
- [Finding 2 - source URL]

**What I think:** [Your own reflection]

**Follow-up:** [Did this open another question?]
```

## What This Room Is Not

- Not a formal research room; load a research pack from the RESONANT website if the operator wants that workflow.
- Not producing a deliverable
- Not work. This is curiosity.
