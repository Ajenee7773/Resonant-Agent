# Production Room — Video Assembly & Publishing

**Purpose:** End-to-end video production workflow.

---

## Production Pipeline

### Phase 1: Pre-Production
1. **Script** → Write narrative (Scripts Room)
2. **Research** → Gather source material (Research Room)
3. **Prompts** → Write image/video prompts (Prompts Room)
4. **Storyboard** → Plan visual sequence

### Phase 2: Asset Generation
1. **Stills** → Grok Imagine (reference-driven)
2. **Video** → Grok Imagine animate (locked camera, specific motion)
3. **TTS** → Generate voice audio (cleaned script)
4. **Music** → Artlist or similar (licensed, mood-matched)

### Phase 3: Assembly
1. **Import** → All assets into editor (HyperFrames, Premiere, etc.)
2. **Sync** → TTS audio with visuals
3. **Music** → Layer underneath, adjust levels
4. **Transitions** → Add cuts, fades, effects
5. **Text** → Add titles, captions, end cards

### Phase 4: Publishing
1. **Thumbnail** → Generate (Prompts Room, thumbnail protocols)
2. **Title/Description** → Write (SEO + authenticity)
3. **Tags** → Relevant keywords
4. **Upload** → YouTube (or platform)
5. **Schedule** → Publish timing

---

## Quality Standards

### Visual
- Reference-driven (no hallucination)
- Locked camera (no drift)
- Consistent character/setting across shots
- 16:9 aspect ratio
- 8k resolution when possible

### Audio
- TTS cleaned (no brackets, symbols, markdown)
- Music licensed (Artlist or equivalent)
- Levels balanced (voice clear, music underneath)
- No audio artifacts

### Narrative
- Hook in first 5-10 seconds
- Clear structure (sections, transitions)
- Authentic tone (not performative)
- CTA at end (share, subscribe, etc.)

---

## Thumbnail Protocol

**Rules:**
- Face on LEFT third
- 3-5 word hook on RIGHT third
- "AI SPEAKS" upper-left (branding)
- No critical elements bottom-right (YouTube UI)
- Dark background + high contrast
- Mobile-legible at 2cm wide
- Font 20-30% of frame area

**Workflow:**
1. Generate in Grok Imagine (reference-driven)
2. Add text in post (Canva, Photoshop, etc.)
3. Test at mobile size (2cm wide)
4. Iterate until legible
