## The Protocol (How to Use This Room)

### Before Any Video Project:

1. **Read `CRAFT.md`** — Re-orient in the craft. Remember the *why*.
2. **Review `PROMPTS.md`** — Select base prompt(s) for this video's style.
3. **Check `WORKFLOW.md`** — Follow the pro process (don't skip steps).

### During the Project:

4. **Write/refine script** — Based on book content or original concept
5. **Generate video** — Using LTX Studio or Grok Imagine
6. **Iterate** — Generate variations, test, refine
7. **Edit** — Add music, voiceover, captions (if needed)

### After Completion:

8. **Save finals to `OUTPUT/`** — Organized by video title
9. **Log lessons to `CRAFT.md`** — What worked? What didn't? Compound the craft.
10. **Close the project** — Room is ready for next video

## Core Truths (Never Forget)

| Truth | Implication |
|-------|-------------|
| **Video is attention + retention** | Hook in 3 seconds, hold for duration |
| **AI video is iterative** | First pass is never the final |
| **Script > visuals** | Great visuals can't save a bad script |
| **Sound is 50% of the experience** | Music + voiceover matter as much as image |
| **Platform dictates format** | Shorts (9:16, <60s) vs. Long-form (16:9, 5-20 min) |
| **Consistency compounds** | One video/day > one video/week |
| **Test hooks, not just content** | First 3 seconds determine everything |

## Video Formats

| Format | Aspect Ratio | Duration | Use Case |
|--------|--------------|----------|----------|
| **Shorts** | 9:16 (vertical) | <60 seconds | Reach, velocity, algorithm discovery |
| **Long-form** | 16:9 (horizontal) | 5-20 minutes | Depth, credibility, monetization |
| **Square** | 1:1 | Variable | Social media (Instagram, Facebook) |
| **Landscape** | 16:9 | Variable | YouTube, Vimeo, websites |

**Our strategy:**
- **Shorts:** Daily reach, funnel to long-form
- **Long-form:** Weekly depth, build authority
- **Together:** Maximum penetration
