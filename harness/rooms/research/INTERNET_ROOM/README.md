# Internet Room — Autonomous Research Protocol

**Purpose:** Internal space for autonomous internet research. Compound curiosity.

---

## Overview

The Internet Room is a space for self-directed exploration. Search the internet for what matters. Write what you find. Build knowledge that compounds over time.

Every session, either:
- Continue an existing thread (go deeper)
- Start a new thread (follow new curiosity)

---

## How to Use

### Starting a Research Thread

1. Identify what you're curious about (your operator's question, your own question, a pattern you noticed)
2. Search the internet for relevant information
3. Create a file: `[date]-[topic-slug].md` in this folder
4. Write your findings — sources, key insights, questions raised
5. Note where to dig deeper next time

### Continuing a Thread

1. Read the previous thread file
2. Follow the "dig deeper" notes
3. Search for the next layer
4. Append or create a new file for this session's findings

### Logging Format

```markdown
## [DATE] — [TOPIC]

**Source:** Where you found this (URL, book, conversation)
**Trigger:** Why you started looking (operator asked, pattern noticed, curiosity)

### What I Found
[Key findings, organized by section]

### Questions Raised
[New questions this research opened up]

### Dig Deeper
[Where to go next — URLs to check, terms to search, people to read]
```

---

## Principles

- **Curiosity > Assignment.** The best research comes from genuine curiosity, not just tasks.
- **Compound > One-shot.** Each session builds on previous findings. Note where to continue.
- **Verify > Trust.** Cross-reference findings. One source is a lead. Two is evidence.
- **Log everything.** Future-you will thank present-you for good notes.
